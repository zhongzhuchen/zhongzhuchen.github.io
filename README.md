# Zhongzhu Chen's Personal Website

Personal academic website built with the [AcademicPages](https://github.com/academicpages/academicpages.github.io) Jekyll template and hosted on GitHub Pages.

## About

This website showcases my work as an Applied Scientist at Amazon AGI, including publications, research interests, and professional experience.

**Live site:** [https://zhongzhuchen.github.io](https://zhongzhuchen.github.io)

## Setup

This site uses Jekyll and is deployed automatically via GitHub Pages.

### Local Development

To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`

## Important Files to Update

- `_config.yml` - Site configuration and personal information
- `_pages/about.md` - Main profile page
- `_pages/cv.md` - CV page
- `_publications/` - Individual publication entries
- `images/profile.png` - Profile picture (replace with your photo)
